We also learned the uses of Getters and Setters which are public methods used to access and modify the private fields (variables) of a class. However this works by acting as controlled access points that allow external classes to interact with private fields through public methods. This concept, known as encapsulation, ensures that the internal data of an object is hidden from direct external modification.
